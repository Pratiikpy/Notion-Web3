🐱 Custom Cursor Setup for Irys Snippet Vault

I've implemented the custom cursor functionality for your website! Here's how to add your cute cat cursor images:

📁 Step 1: Prepare Your Cursor Images

You'll need to create 4 versions of your cat cursor image:

  • cursor-cat.png - Default cursor (32x32px recommended)
  • cursor-cat-pointer.png - For buttons and links (32x32px)
  • cursor-cat-text.png - For text input areas (32x32px)
  • cursor-cat-wait.png - For loading states (32x32px)

📂 Step 2: Add Images to Your Project

Place all cursor images in the /app/frontend/public/ folder:

/app/frontend/public/ ├── cursor-cat.png ├── cursor-cat-pointer.png ├── cursor-cat-text.png └── cursor-cat-wait.png

🎨 Step 3: Image Requirements

  • Size: 32x32 pixels (recommended) or 16x16 pixels
  • Format: PNG with transparency
  • Hotspot: The cursor hotspot should be at 16,16 (center) for 32x32 images

✨ Step 4: What's Already Implemented

I've already added custom cursor styles for:

  • 🖱️ Default cursor throughout the app
  • 👆 Pointer cursor for buttons and links
  • 📝 Text cursor for input fields and text areas
  • ⏳ Wait cursor for loading states
  • 🎯 Hover effects on interactive elements

🔧 Step 5: Test Your Cursors

Once you add the images, refresh your website and test:

  • Move mouse around the page (should show your cat cursor)
  • Hover over buttons (should show pointer version)
  • Click in text inputs (should show text version)
  • Hover over snippet cards and tags (interactive cursors)
💡 Pro Tip: If you only have one cat image, you can use the same image for all cursor states by copying it 4 times with different names. The CSS will handle the rest!

🎮 Advanced: CSS Cursor Mapping

The cursor changes are mapped as follows:

/* Default cursor everywhere */ cursor: url('/cursor-cat.png') 16 16, auto; /* Buttons, links, clickable elements */ cursor: url('/cursor-cat-pointer.png') 16 16, pointer; /* Text inputs, text areas */ cursor: url('/cursor-cat-text.png') 16 16, text; /* Loading/disabled states */ cursor: url('/cursor-cat-wait.png') 16 16, wait;

Your cute cat cursor is ready to charm all your website visitors! 🐱✨
